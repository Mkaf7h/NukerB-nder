## Overview

**NUkerBinder** هو أداة احترافية لدمج الملفات وتشفيرها (Binding & Crypting) مصممة للأداء العالي والتخفي. تتيح لك دمج ملفات متعددة في ملف تنفيذي واحد (.exe) يعمل بشكل مستقل. تمت برمجة الأداة بلغة **C** باستخدام **Win32 API** الصرفة لضمان عدم وجود اعتمادات خارجية، وأفضل أداء ممكن، وواجهة مستخدم خفيفة وكلاسيكية.

##  المميزات المتقدمة (New Options)

| الخيار | الوصف التقني |
| :--- | :--- |
| **📦 Self-Extracting EXE** | توليد ملف EXE واحد يقوم بفك الملفات وتشغيلها تلقائياً في الذاكرة أو القرص. |
| **🛡️ Watchdog Engine** | محرك مراقبة يعمل في الخلفية لقتل أدوات التحليل والـ Debuggers فور تشغيلها. |
| **💀 Anti-Kill (Critical)** | تعيين العملية كـ "Critical Process"، مما يؤدي لظهور الشاشة الزرقاء (BSOD) إذا حاول أحد قتلها. |
| **🚫 Host Blocker** | حظر أكثر من 20 موقعاً لشركات الحماية (مثل VirusTotal، Kaspersky، إلخ) عن طريق ملف الـ Hosts. |
| **☣️ Anti-VM BSOD** | فحص متقدم للبيئات الوهمية (VMware, VirtualBox) وإيقاف تشغيل الجهاز أو تجميده عند كشفها. |
| **⚡ Legendary BAT** | توليد ملف Batch مشفر بتقنية `certutil` و `^` obfuscation لتجاوز الحمايات. |
| **🔓 UAC Bypass** | تخطي شاشة تحكم حساب المستخدم (UAC) باستخدام ثغرة `fodhelper` للحصول على صلاحيات أدمن. |
| **🔐 FUD Encryption** | تشفير الملفات المدمجة بخوارزمية **RC4** لضمان عدم كشف محتواها من قبل الماسحات الضوئية. |
| **💉 Manifest Injection** | حقن ملف Manifest لطلب صلاحيات الأدمن تلقائياً أو العمل بصلاحيات المستخدم الحالي. |
| **🗜️ RLE Compression** | محرك ضغط داخلي لتقليل حجم الملف النهائي بشكل كبير. |
| **📍 Multi-Drop Paths** | تحديد مسار فك مستقل لكل ملف (AppData, Temp, System32, Current, Custom). |
| **🔄 Persistence** | تثبيت الملف في النظام ليعمل تلقائياً مع بداية التشغيل عبر الـ Registry أو Task Scheduler. |
| **🔥 Self-Delete (Melt)** | ميزة الانتحار: الملف التنفيذي يحذف نفسه فور الانتهاء من العمل لإزالة الأثر. |

##  Usage (طريقة الاستخدام)

1. **إضافة الملفات**: قم بسحب وإفلات الملفات داخل القائمة أو اضغط كليك يمين واختر **Add Files**.
2. **إعدادات الملفات**: اضغط كليك يمين على أي ملف لتحديد مسار النزول (Drop Path) ووضع التشغيل.
3. **التشفير والحماية**: اضغط على زر **CRYPTER** لتفعيل خيارات التخفي مثل Anti-VM و Anti-Debug.
4. **الأيقونة**: اضغط على **ICON** لاختيار أيقونة احترافية لملفك.
5. **البناء**:
   - اضغط **BUILD EXE** للحصول على ملف تنفيذي احترافي.
   - اضغط **BUILD BAT** للحصول على ملف Batch مشفر (FUD).
